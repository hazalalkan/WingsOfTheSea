# WingsOfTheSea
Wings of the Sea is a 3D Processing adventure set in a magical universe. Rosa, a fairy searching for her lost bunny Lumi, journeys through an enchanted forest and a mystical lake. Along the way, she shifts between fairy and mermaid forms, avoids obstacles, collects magical petals and pearls, and races toward her fate.

# How to Run
1. Install Processing.
2. Open WingsOfTheSea.pde in Processing.
3. Install the Processing Sound library:
   - Open Processing.
   - Go to Sketch > Import Library > Manage Libraries...
   - Search for Sound.
   - Install the official Sound library by The Processing Foundation.
4. Press the Run button.

- The game should run on macOS and Windows if the Sound library and all assets are installed correctly.
- If sound does not work, first check that the Processing Sound library is installed.
- For the best experience, play with sound enabled. Have fun!
